# 首次启动与第一条对话

## 目标

配好一个模型供应商，发一条消息，拿到回复。

## 步骤 1：准备 API Key

在你选择的模型供应商后台创建 API Key，复制保存好。
建议单独为 BotHub 建一组 Key，方便后续轮换。

## 步骤 2：添加供应商

1. 打开 BotHub，进入 `设置` > `模型服务`。
2. 点击 `添加厂商`。
3. 填写：
   - `厂商名称`（随便起个名字，方便你认）
   - `基础地址`（Base URL，供应商给的接口地址）
   - `API Key`
4. 保存后，进入该厂商的模型列表，添加至少一个模型（填 Model ID）。

## 步骤 3：选择模型

回到对话页面，在模型选择区选一个刚配好的模型。

## 步骤 4：发消息

输入一条简单消息，比如：`你好，用一句话介绍你自己。`
收到正常回复就算配置成功。

## 成功判定

- 没有认证错误（401/403）
- 没有模型不存在错误（404 或模型名错误）
- 正常收到文本回复

## 配置好之后建议

- 再加一个备用模型，以防单一供应商挂了
- 记好你在用的 Model ID，避免拼写错误
