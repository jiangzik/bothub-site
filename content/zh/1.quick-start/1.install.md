# 安装与下载

## 平台状态

| 平台 | 状态 | 下载入口 |
| --- | --- | --- |
| macOS | 可用 | 首页下载按钮 |
| Windows | 可用 | 首页下载按钮 |
| Linux | 可用 | 首页下载按钮 |
| Android | 可用 | 首页下载按钮 |
| iOS | 开发中 | 暂无公开安装包 |

## Desktop 安装步骤

1. 打开官网首页，点击对应平台下载按钮。
2. macOS 使用 `.dmg`，Windows 使用 `.exe`，Linux 使用 `.AppImage`。
3. 按系统安装向导完成安装。
4. 首次启动时如出现系统安全提示，请允许 BotHub 运行。

## Android 安装步骤

1. 在安卓设备打开官网并下载 APK。
2. 首次安装 APK 时，按系统提示允许“安装未知应用”。
3. 安装完成后启动 BotHub。

## 安装成功判定

- 应用可以正常启动并进入主界面
- 页面可见模型或会话相关入口
- 不闪退、不立即卡死

## 安装失败时先检查

- 安装包是否完整下载
- 系统版本是否过旧
- 系统是否拦截了未签名或未知来源应用
- 是否有安全软件阻止安装

如果仍无法解决，建议先查看 [常见问题（FAQ）](/zh/faq/overview)。
