# 安装与下载

## 平台状态

| 平台 | 状态 | 下载方式 |
| --- | --- | --- |
| macOS | 可用 | 首页下载 `.dmg` |
| Windows | 可用 | 首页下载 `.exe` |
| Android | 可用 | Google Play 或首页下载 APK |
| iOS | 开发中 | 暂无 |

## Desktop 安装步骤

1. 打开官网首页，点击对应平台下载按钮。
2. macOS 打开 `.dmg` 拖入应用文件夹，Windows 运行 `.exe` 安装程序。
3. 首次启动时如出现系统安全提示，请允许 BotHub 运行。

> **macOS 提示"已损坏，无法打开"？** BotHub 当前尚未完成 Apple 签名，并非真的损坏。详细放行步骤见 [macOS 安装指南](/zh/quick-start/macos-install)。

## Android 安装步骤

1. 推荐从 Google Play 安装。
2. 也可以在首页下载 APK 直接安装，首次需要按系统提示允许"安装未知应用"。
3. 安装完成后启动 BotHub。

## 安装成功判定

- 应用正常启动，看到对话界面
- 没有闪退或白屏

## 安装失败时先检查

- 安装包是否下载完整
- 系统版本是否过旧
- 系统是否拦截了未知来源应用
- 是否有安全软件阻止安装

如果仍然解决不了，看 [常见问题](/zh/faq/overview)。
