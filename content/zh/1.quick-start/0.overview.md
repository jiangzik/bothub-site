# 快速开始

如果你是第一次使用 BotHub，按下面顺序阅读即可。

1. [安装与下载](/zh/quick-start/install)
2. [首次启动与第一条对话](/zh/quick-start/first-run)

## 你将完成的目标

- 在 Desktop 或 Android 成功安装 BotHub
- 完成模型供应商配置
- 成功发出第一条消息并拿到回复

## 前置准备

- 稳定网络环境
- 至少一个可用模型供应商账号
- 对应供应商的 API Key

## 建议

首次配置建议优先在 Desktop 完成，界面信息更完整，排查也更方便。
