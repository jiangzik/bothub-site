# 常见问题（FAQ）

## 1. 下载速度慢或下载失败怎么办？

- 更换网络后重试
- 关闭可能拦截下载的安全软件
- 确认磁盘空间充足
- 优先在桌面浏览器中下载，不建议在受限内置浏览器下载

## 2. 安装后打不开应用怎么办？

- macOS 检查“隐私与安全性”是否阻止运行
- Windows 以普通用户启动失败时，可尝试管理员权限
- Linux 检查执行权限（尤其是 AppImage）

## 3. 为什么模型返回 401/403？

通常是 API Key 无效、过期或权限不足。  
请重建 Key 并确认绑定的组织/项目权限。

## 4. 为什么提示模型不存在或 404？

模型名称（Model ID）与供应商后台不一致。  
请直接复制后台模型名，避免手工输入拼写错误。

## 5. 为什么有时响应很慢？

- 供应商侧拥堵
- 当前模型本身较慢
- 提示词过长导致输入 token 过多

建议先用小任务验证链路，再切回目标模型。

## 6. 为什么同样问题每次结果差异很大？

- 温度参数偏高
- 提示词约束不够明确
- 会话上下文混入了旧任务内容

建议降低温度并新建会话重试。

## 7. 升级版本后如何快速自检？

按以下顺序做三条测试：

1. 普通问答（检查可用性）
2. JSON 输出（检查格式稳定性）
3. 长文总结（检查长输出稳定性）

任一项异常，先切换备用模型确认是配置问题还是版本问题。

## 8. iOS 什么时候可用？

iOS 当前处于开发中，建议关注官网更新与社区通知。
