# 常见问题（FAQ）

## 1. 下载速度慢或下载失败？

- 换个网络试试
- 关掉可能拦截下载的安全软件
- 确认磁盘空间够
- 用桌面浏览器下载，别在微信等内置浏览器里下

## 2. 安装后打不开？

- macOS：去"隐私与安全性"里允许运行
- Windows：试试管理员权限启动

## 3. 模型返回 401/403？

API Key 无效、过期或权限不够。去供应商后台重新建 Key，检查有没有绑对组织/项目。

## 4. 提示模型不存在或 404？

Model ID 和供应商后台不一致。直接从后台复制模型名，别手打。

## 5. 响应很慢？

- 供应商那边拥堵
- 用的模型本身就慢
- 提示词太长，输入 token 太多

先用一条短消息试，确认是不是模型本身的问题。

## 6. 同样的问题每次结果差很远？

- Temperature 太高
- 提示词约束不够具体
- 对话上下文混了旧内容

调低 Temperature，新建对话重试。

## 7. 升级后怎么快速验证？

发三条测试消息：

1. 简单问答
2. 要求输出 JSON
3. 长文总结

有一条不对，先切备用模型看是配置问题还是版本问题。

## 8. MCP 服务器连不上？

- 检查服务器地址是否正确
- 点"测试连接"看具体报错
- 确认 MCP Server 正在运行且端口没被防火墙拦

## 9. 定时任务没跑？

- 检查任务是否启用
- 桌面端需要保持运行，关了就不会触发
- 看运行历史里有没有报错记录

## 10. iOS 什么时候出？

还在开发中，关注官网和社区通知。
