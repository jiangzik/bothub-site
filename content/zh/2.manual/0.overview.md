# 使用手册

已经装好、能正常聊天了？下面是各功能的详细说明。

## 基础功能

1. [供应商与模型配置](/zh/manual/providers) — 添加厂商、管理模型、调参数
2. [会话与上下文管理](/zh/manual/chat-and-context) — 对话组织、群聊、项目

## 当前支持的主要功能

- **多模型切换** — 对话中间随时换模型，每个 Bot 可绑不同模型
- **MCP 工具** — 接入 MCP Server，让模型读写文件、控浏览器、搜网页
- **技能** — 把常用操作写成可复用的技能，支持从 GitHub 导入
- **定时任务** — 设好时间让 Bot 自动跑任务（桌面端）
- **群聊** — 多个 Bot 在同一对话中各自回复
- **Bot 集成** — 桌面端可接飞书、Telegram、QQ 机器人
- **远程控制** — 手机连电脑，远程终端和屏幕控制
- **联网搜索** — 支持 Bing、Exa、Tavily、Brave、SearXNG 等后端
- **执行策略** — 控制工具执行权限、审批模式，可选 Docker 沙箱
