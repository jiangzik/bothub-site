# 使用手册

1. [供应商与模型配置](/zh/manual/providers) — 添加厂商、管理模型、调参数
2. [会话与上下文管理](/zh/manual/chat-and-context) — 对话组织与项目

## 当前支持的主要功能

- **多模型切换** — 对话中间随时换模型，每个 Bot 可绑不同模型
- **MCP 工具** — 接入 MCP Server，让模型读写文件、控浏览器、搜网页
- **技能** — 把常用操作写成可复用的技能，支持从 GitHub 导入
- **知识库 RAG** — 上传 PDF / DOCX / 图片 / 网页，向量检索 + 语义重排自动注入上下文（桌面端 & Android）
- **定时任务** — 设好时间让 Bot 自动跑任务（桌面端）
- **IM 机器人桥接** — 桌面端 / Android 端均可接飞书、Telegram、QQ、微信
- **远程控制** — 手机连电脑，远程终端和屏幕控制
- **内置 Ubuntu 24.04** — 手机上的本地 Linux 环境，AI 可直接在设备上执行代码，无需外接电脑（Android）
- **TTS 朗读 & 声音克隆** — 云端 TTS 模型朗读回复，可克隆个人音色作为专属预设（桌面端 & Android，Pro）
- **联网搜索** — 支持 Bing、Exa、Tavily、Brave、SearXNG 等后端
- **执行策略** — 控制工具执行权限、审批模式，可选 Docker 沙箱
