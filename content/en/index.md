<section id="features" class="bothub-section">
<div class="bothub-container">
<div class="bothub-section-head">
<div class="bothub-section-eyebrow">Built for power users</div>
<h2>One client, every AI tool you need</h2>
</div>
<div class="bothub-features">
<div class="bothub-feature">
<div class="bothub-feature-icon"><bothub-icon name="i-lucide-sparkles"></bothub-icon></div>
<h3>Sign in to top AI models</h3>
<p>Sign in once and you're talking to the world's top AI models on a single account — no provider keys to apply for. Free credits on signup, so you can start building right away.</p>
<div class="bothub-feature-meta">free credits · top models</div>
</div>
<div class="bothub-feature">
<div class="bothub-feature-icon"><bothub-icon name="i-lucide-key-round"></bothub-icon></div>
<h3>Bring your own key</h3>
<p>Drop in your own API keys — requests go straight to the model provider. Any OpenAI-compatible or local endpoint works too. Multi-key rotation, pay by usage, no middleman.</p>
</div>
<div class="bothub-feature">
<div class="bothub-feature-icon"><bothub-icon name="i-lucide-shuffle"></bothub-icon></div>
<h3>Switch models anytime</h3>
<p>Swap models mid-conversation without starting over. Each bot can bind to a different model — match the model to the task.</p>
</div>
<div class="bothub-feature">
<div class="bothub-feature-icon"><bothub-icon name="i-lucide-wrench"></bothub-icon></div>
<h3>Tool use, built in</h3>
<p>Bash, file I/O, browser automation, screenshot, clipboard, system info. Need more? Plug in any MCP server.</p>
<div class="bothub-feature-meta">desktop-only · macOS · Windows</div>
</div>
<div class="bothub-feature">
<div class="bothub-feature-icon"><bothub-icon name="i-lucide-library"></bothub-icon></div>
<h3>Knowledge base RAG</h3>
<p>Upload PDFs, DOCX, XLSX, images, or web pages. Vector search + semantic rerank + entity graph, injected into context automatically.</p>
</div>
<div class="bothub-feature">
<div class="bothub-feature-icon"><bothub-icon name="i-lucide-puzzle"></bothub-icon></div>
<h3>Skills & MCP</h3>
<p>Save workflows as skills or import from GitHub. Connect any MCP server, with per-tool approval policies.</p>
</div>
<div class="bothub-feature">
<div class="bothub-feature-icon"><bothub-icon name="i-lucide-clock"></bothub-icon></div>
<h3>Scheduled tasks</h3>
<p>Cron or countdown. The bot runs on its own. Results flow back to the chat, or out to Feishu / Telegram.</p>
</div>
<div class="bothub-feature">
<div class="bothub-feature-icon"><bothub-icon name="i-lucide-message-square-share"></bothub-icon></div>
<h3>IM bot bridges</h3>
<p>Push your chats to Feishu, Telegram, QQ, or WeChat — every message routes through BotHub. Native on desktop, also available on Android.</p>
<div class="bothub-feature-meta">desktop · android</div>
</div>
<div class="bothub-feature">
<div class="bothub-feature-icon"><bothub-icon name="i-lucide-image"></bothub-icon></div>
<h3>Image generation, included</h3>
<p>Mainstream text-to-image models, all in the chat. Save to disk, attachments, or your knowledge base. Iterate with "different style / different ratio".</p>
<div class="bothub-feature-meta">image_gen · built-in</div>
</div>
</div>
