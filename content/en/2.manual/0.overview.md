# User Manual

Already installed and chatting? Here's what each feature does.

## Basics

1. [Provider and Model Configuration](/en/manual/providers) — Add providers, manage models, adjust parameters
2. [Conversation and Context Management](/en/manual/chat-and-context) — Organize chats, group chats, projects

## Features overview

- **Multi-model switching** — Switch models mid-conversation; each bot can use a different model
- **MCP tools** — Connect MCP servers so models can read/write files, control a browser, search the web
- **Skills** — Save reusable multi-step operations as skills; import from GitHub
- **Scheduled tasks** — Set bots to run automatically on a schedule (desktop)
- **Group chats** — Multiple bots in one conversation, each responding independently
- **Bot integrations** — Connect Feishu, Telegram, or QQ bots on desktop
- **Remote control** — Connect your phone to your PC for remote terminal and screen control
- **Web search** — Supports Bing, Exa, Tavily, Brave, SearXNG, and more
- **Execution policy** — Control tool permissions, approval modes, optional Docker sandbox
