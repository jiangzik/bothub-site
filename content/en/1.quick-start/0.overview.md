# Quick Start

If this is your first time using BotHub, follow this order.

1. [Install and Download](/en/quick-start/install)
2. [First Launch and First Message](/en/quick-start/first-run)

## What you will achieve

- Install BotHub on Desktop or Android
- Configure a model provider
- Send your first message and get a valid response

## Prerequisites

- Stable network connection
- At least one model provider account
- A valid API key for that provider

## Recommendation

Do your first setup on Desktop for easier troubleshooting and clearer settings visibility.
