# Install and Download

## Platform availability

| Platform | Status | Download method |
| --- | --- | --- |
| macOS | Available | `.dmg` from homepage |
| Windows | Available | `.exe` from homepage |
| Android | Available | Google Play or APK from homepage |
| iOS | In development | Not yet available |

## Desktop installation

1. Open the homepage and click your platform's download button.
2. macOS: open the `.dmg` and drag to Applications. Windows: run the `.exe` installer.
3. If the OS shows a security prompt on first launch, allow BotHub to run.

> **macOS shows "is damaged, can't be opened"?** BotHub is not yet signed with an Apple Developer ID — it's not actually damaged. See the [macOS Install Guide](/en/quick-start/macos-install) for step-by-step workarounds.

## Android installation

1. Google Play is recommended.
2. You can also download the APK from the homepage. Allow "Install unknown apps" when prompted.
3. Launch BotHub after installation.

## Success criteria

- The app launches and shows the conversation screen
- No crash or blank screen

## If installation fails

- Check whether the package downloaded completely
- Verify your OS version is supported
- Check whether the system blocked unknown apps
- Check whether antivirus software blocked the installer

If the issue persists, see [FAQ](/en/faq/overview).
