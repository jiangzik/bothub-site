# Install and Download

## Platform availability

| Platform | Status | Download Entry |
| --- | --- | --- |
| macOS | Available | Home page download button |
| Windows | Available | Home page download button |
| Linux | Available | Home page download button |
| Android | Available | Home page download button |
| iOS | In development | No public package yet |

## Desktop installation steps

1. Open the official home page and click your platform button.
2. Use `.dmg` for macOS, `.exe` for Windows, and `.AppImage` for Linux.
3. Complete the system installer flow.
4. If the OS shows a security prompt, allow BotHub to run.

## Android installation steps

1. Open the official site on your Android device and download the APK.
2. Allow "Install unknown apps" when prompted.
3. Launch BotHub after installation.

## Success criteria

- The app launches and reaches the main screen
- You can see conversation/model related entries
- No immediate crash or freeze

## If installation fails

- Check whether the package downloaded completely
- Verify your OS version is supported
- Check whether OS security blocked unknown apps
- Check whether antivirus/security tools blocked the installer

If the issue persists, continue with the [FAQ](/en/faq/overview).
